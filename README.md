# WheelsMD
Automotive calculators and conversion tools for Palm OS PDAs. Made with NSBasic IDE.

Wheels MD is a simple and intuitive set of calculators and conversion tools for automotive repairs and testing. Calculate horsepower, displacement, mean piston speed, compression ratios, fuel economy, tire diameter, speedometer margin of error, amperage draw, and coolant/fluid ratios. You can also convert torque and volume data to different units. If you're a fan of cars, this app might just come in handy.

NOTE: You need to install the Palm OS MathLib for this app to be fully functional.
